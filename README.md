# Simi - Versión PowerShell (rama archivada)

Esta rama está archivada y no se mantiene ni recibirá actualizaciones.

## Propósito

La rama `archive-powershell` existe únicamente como archivo histórico ya que Simi inicialmente fue escrito en PowerShell, solo compatible con Windows. Ahora el proyecto es multiplataforma, por lo que esta versión no forma parte del flujo principal del proyecto.

## Importante

- No se debe desarrollar en esta rama
- No se deben abrir pull requests hacia `main`
- No se garantiza compatibilidad ni soporte
- No se aceptan cambios salvo por razones históricas

## Ramas activas del proyecto

- **`main`** → proyecto activo
- **`archivos-idioma`** → archivos de idioma
